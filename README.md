PlacementPrep AI

AI-powered platform to automate company research and placement preparation.

Goal

Reduce the time students spend researching companies and provide personalized preparation insights.

Tech Stack (V1)

* Frontend: Next.js + Tailwind CSS
* Backend: FastAPI (Python)
* AI: LLM APIs (yet to decide), LangChain
* Web Scraping: BeautifulSoup
* Database: Supabase (PostgreSQL)
